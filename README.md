# spy-sample

Scrapy Learning Program.


## Python version:  Python 3.7 or above

Install virtual enviroment:

```bash
$ python3 -m venv venv
```

Activate venv:

```bash
$ . venv/bin/activate
$ cd [project_dir]
$ scrapy [command ...]
```

## Sample Projects:


- **spytest**

    - 游동 cptrack

        ```bash
        $ scrapy crawl --nolog cptrack -a num=FX000090696630220
        ```

    - 游동 tttrack

        ```bash
        $ scrapy crawl --nolog tttrack -a num=MHE827061910013817 -o tttrack.csv
        ```
